# 🚀 Azure Active Directory VM Lab (Terraform)

## 📌 Overview
This project demonstrates how to deploy an Azure Virtual Machine using **Terraform (Infrastructure as Code)**.  
The lab simulates a basic environment that can be used for **Active Directory or domain services testing**.

---

## 🧰 Technologies Used
- Terraform
- Microsoft Azure
- Azure CLI
- Visual Studio Code
- Windows PowerShell

---

## 🏗️ Architecture
The Terraform configuration deploys:
- Resource Group
- Virtual Network (VNet)
- Subnet
- Network Interface (NIC)
- Public IP Address
- Windows Virtual Machine
